HTML Tags VS elements
---------------------
- HTML tag is anything that is INSIDE an pair of angle brackets < >
    - It has an
        - Opening tag < > 
        - Closing tag </ >


- HTML element refers to BOTH 
    - Opening and closing HTML tags
    - Content embedded within the HTML tags 


Example:
<!-- An <h1> HTML element  -->
<h1> </h1>

<!-- An <h1> HTML element  -->
<h1>Hello there</h1>
