Horizontal Rule 
---------------
- The HTML element <hr> represents a thematic break between paragraph-level elements
    - It is a void element 
    - It is visually represented by a horizontal line 

    - Reference (https://developer.mozilla.org/en-US/docs/Web/HTML/Element/hr)

    - Use case:
        - To implement a shift of topic within a section
        - To implement a change of scene in a story


Syntax:

<hr>
