Paragraph Element
-----------------
- The HTML element <p> represents a paragraph
    - It formats and place content in a HTML document
        - Often in the form of blocks of text
        - Separated from adjacent blocks by blank lines

    - It is a block-level element 

    - Reference (https://developer.mozilla.org/en-US/docs/Web/HTML/Element/p)


Example:

<p>This is a paragraph</p>
