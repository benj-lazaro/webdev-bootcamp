CSS Colours
-----------
- The CSS property "color"
    - Sets the color of a targeted CSS Selector

    Syntax:

    <CSS_Selector> {
        color: <color_value>;
    }


- The color values are represented using
    - Named colors
        - Using named colors
        - e.g. red, blue, yellow
        - Reference (https://developer.mozilla.org/en-US/docs/Web/CSS/named-color)

    - Hex colors
        - Represented by 3 fractions of Red, Green & Blue
        - Prefixed with a "#"
        - Followed by hexedecimal numbers
        - e.g. #5D3891 (5D of Red, 38 of Green and 91 of Blue )
        - Reference (https://developer.mozilla.org/en-US/docs/Web/CSS/hex-color)

    - Color values are represented in a number of ways
        - Named colors
        - Hex code color


- Free online color tools
    - ColorHunt.co
        - Color palette tool
        - Website (https://colorhunt.co/)

    - RGB Mixer 
        - Color mixer tool
        - Offer color values in Decimal and/or Hexidecimal format
        - Website (https://www.csfieldguide.org.nz/en/interactives/rgb-mixer/)
