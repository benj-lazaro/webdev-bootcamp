CSS Colours
-----------
- CSS color properties sets the color of a targeted CSS Selector
    - Color values in CSS can be represented in a number of ways

Syntax
------
<css_selector> {
    color: <color_value>;
}


- Named color
    - Color is represented by its given name (e.g. black, red, cornflowerblue, etc.)
    - Reference (https://developer.mozilla.org/en-US/docs/Web/CSS/named-color)

    <css_selector> {
        color: red;
    }


- Hex Code / Hex Color 
    - Color is represented by a hexedecimal value
    - Formed by three (3) fractions of Red, Blue & Green
    - Prefixed by a "#"
    - Reference (https://developer.mozilla.org/en-US/docs/Web/CSS/hex-color)

    <css_selector> {
        color: #5D3891;     /* consists of hex code red: 5D, green: 38, blue: 91 */
    }
