The List Element
----------------
- HTML has two (2) types of list
    - Ordered list
        - The HTML element <ol>...</ol> represents an ordered list of items
        - Rendered as a numbered list
        - Reference (https://developer.mozilla.org/en-US/docs/Web/HTML/Element/ol)

    - Unordered list
        - The HTML element <ul>...</ul> represents an unordered list of items
        - Rendered as a bulleted list
        - Reference (https://developer.mozilla.org/en-US/docs/Web/HTML/Element/ul)


- Each item inside a list is represented by
    - Each HTML element <li>...</li> MUST be contained within a parent list
        - Be it an Ordered or Unordered list

    - Reference (https://developer.mozilla.org/en-US/docs/Web/HTML/Element/li)

    - Example:

        <ol>
            <li>first item</li>
            <li>second item</li>
            <li>third item</li>
        </ol>

        <ul>
            <li>first item</li>
            <li>second item</li>
            <li>third item</li>
        </ul>
