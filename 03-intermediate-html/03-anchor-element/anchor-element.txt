Anchor Elements
---------------
- The <a> element along with the attribute "href" creates a hyperlink to:
    - An email address
    - A file 
    - A separate HTML document
    - A location within the same HTML document
    - a URL address 

    - Reference (https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a)


- The "href" attribute is assigned with a value of a URL
    - This is where a hyperlink points to
