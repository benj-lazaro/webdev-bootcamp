Anchor Elements
---------------
- The HTML element <a> with its attribute "href"
    - Create a hyperlink to another HTML document, file, email or anything else with a URL
    
    - The value assigned to the attribute "href"
        - Refers to the location where the hyperlink points to

    - Reference (https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a)


Example:

<p>You can reach Michael at:</p>

<ul>
  <li><a href="https://example.com">Website</a></li>
  <li><a href="mailto:m.bluth@example.com">Email</a></li>
  <li><a href="tel:+123456789">Phone</a></li>
</ul>
