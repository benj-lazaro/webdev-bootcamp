Project: Portfolio Website
--------------------------
- Create your first web developer portfolio websites
    - Create a new file "index.html"
    - Use HTML boilerplate


- Create a new folder named "public"
    - Store previous HTML projects into this folder
    

- Create a new folder named "assets"
    - Take screenshots of previous HTML projects
    - Store screenshots into this folder


- Link the images and HTML projects stored on the respective sub-folders
    - In the HTML file "index.html"
