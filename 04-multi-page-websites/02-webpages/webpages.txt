What Are Webpages?
------------------
- A multi-paged website comprises of multiple individual web pages
    - With the file "index.html" as the main page


- Each HTML document (i.e. web page) contains specific information
    - Often stored within the same folder
    - But can have their own sub-folder


- In order to link all of the web pages together from the "index.html"
    - The Relative path is used to link them together
